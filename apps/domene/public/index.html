Domain Monitor API Service

Dette er API-tjenesten for Domain Monitor. Bruk nettMonitor frontenden for å administrere dine domener.

Gå til nettMonitor

API Endepunkter

Følgende API-endepunkter er tilgjengelige:

  • GET /api/domains

    Hent alle domener

  • POST /api/domains

    Legg til nytt domene

  • GET /api/domains/:name

    Hent et spesifikt domene

  • DELETE /api/domains/:name

    Slett et domene

  • POST /api/domains/:name/check

    Manuell sjekk av et domene

WebSocket API

WebSocket-API er tilgjengelig på port 3001. Hendelser sendes i sanntid for domeneendringer.
