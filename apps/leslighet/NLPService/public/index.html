API Documentation

All services are available through our RESTful API:

Sentiment Analysis

POST /api/ai/sentiment Content-Type: application/json { "text": "Your text to analyze" }

Summarization

POST /api/ai/summarize Content-Type: application/json { "text": "Your text to summarize" }

Text Correction

POST /api/ai/correct Content-Type: application/json { "text": "Your text to correct" }

AI Chat

POST /api/ai/chat Content-Type: application/json { "messages": [ {"role": "system", "content": "You are a helpful assistant."}, {"role": "user", "content": "Your question here"} ], "model": "gpt-3.5-turbo", "temperature": 0.7, "stream": false }
